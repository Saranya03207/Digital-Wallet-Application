# Digital Wallet Backend

## Features

- User Registration
- Login Authentication
- BCrypt Password Encryption
- Wallet Creation
- Add Money
- Transfer Money
- Withdraw Money
- Balance Check
- Transaction History
- User Block/Activate
- Admin Dashboard
- Swagger API Documentation

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- Spring Security
- MySQL
- Maven
- Swagger OpenAPI

## API Testing

Tested using:
- Postman
- Swagger UI
